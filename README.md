# Forking this website  
Feel free to fork the website to create and deploy a similar webpage for yourself. However, make sure you edit the analytics ID in [`_includes/analytics.html`](_includes/analytics.html) so that your website's views are not logged into my account.

* This site is built using Jekyll. Find the theme here - https://jekyll.github.io/minima/ *
